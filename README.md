# STG
A simple bullet-hell game made using SFML and nlohmann/json

### Controls:
- Movement:  
  - WASD, arrows
  - left joystick, D-pad
- Shoot: 
  - Space
  - A 
- Slow down/show hitbox: 
  - Shift
  - RB
- Pause
  - Esc
  - Start

Only X-input gamepads are supported, mouse is supported only in menus
